STEP 2: TEST RESET (VERY IMPORTANT)
----------------------------------------

POST http://127.0.0.1:8000/reset

No body required

Expected Response:
{
  "observation": {
    "data": {
      "value": 0
    },
    "message": "Current environment state"
  }
}

✔ Check:
- step_count reset to 0
- value initialized
- no error

----------------------------------------
STEP 3: TEST STEP (MAIN FUNCTION)
----------------------------------------

POST http://127.0.0.1:8000/step

Body (JSON):
{
  "action_type": "increase",
  "parameters": {
    "value": 50
  }
}

Expected Response:
{
  "observation": {...},
  "reward": 0.5,
  "done": false,
  "info": {}
}

✔ Check:
- value updated
- reward calculated
- done is false initially

----------------------------------------
STEP 4: MULTIPLE STEPS

    "value": 20
  }
}

✔ Check:
- value keeps increasing
- reward updates correctly
- after max steps → done = true
